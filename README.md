# **English**

# Hammers Galore Mod

This mod adds a variety of practical hammer tools to the game, ranging from basic metal hammers to legendary hammers with unique functions, meeting your various mining and combat needs.

## ⚙️ Configuration Options

You can adjust hammer behavior through the configuration file:

- **Mining Area Size**: Supports 3x3 / 5x5 / 7x7 / 9x9, default is 3x3. Set to `1` to disable area mining.
- **Area Mining Condition**: Choose whether sneaking is required to trigger area mining (enabled by default).
- **Durability Cost**: Set the durability consumed per additional block when area mining (default value is 1).
- **Hunger Cost**: Set whether additional hunger is consumed when area mining (default value is 1).
- **Special Hammer Functions**: Individual effect parameters for special hammers can be adjusted separately.

## 🔨 Basic Hammers

These hammers follow the vanilla tool progression and can be crafted at a crafting table using corresponding materials.

- Wooden Hammer
- Copper Hammer
- Iron Hammer
- Golden Hammer
- Diamond Hammer
- Netherite Hammer

## ✨ Special Hammers
These hammers possess unique mechanisms, bringing more possibilities to gameplay.

**Nether Star Hammer**: When the hammer's durability is low, it will increase attack damage and mining speed;

**Heart of the Sea Hammer**: When the player is in water, they can ignore underwater mining speed penalties, mining as quickly as on land;

**Conduit Hammer**: When the player is in water, they will receive the "Conduit Power" status effect;

**Ender Hammer**: Blocks mined with this hammer will have their drops teleported directly into the player's inventory;

**Magma Hammer**: Mined blocks will drop directly in their smelted form;

**Piston Hammer**: When attacking a target, it applies an extremely high knockback effect;

**Glass Hammer**: Although it has only 1 point of durability, it possesses extremely high attack power. One hit kill, one hit break;

**Sculk Hammer**: Blocks mined with it are directly converted into experience orbs. It is an excellent tool for quickly gaining experience.

**Emerald Hammer**：Grants Luck when mining or attacking based on durability.。

# **中文**

# 锤子多多模组

本模组为游戏添加了多种实用的锤子工具，从基础的金属锤到拥有特殊功能的传奇锤子，满足你的各种挖掘和战斗需求。

## ⚙️ 配置选项

你可以通过配置文件调整锤子的行为：

- **范围挖掘尺寸**：支持 3x3 / 5x5 / 7x7 / 9x9，默认开启 3x3。设置为 `1` 可禁用范围挖掘。
- **范围挖掘条件**：可选择是否需要潜行时才能触发范围挖掘（默认开启）。
- **耐久消耗**：设置范围挖掘时，每个额外方块消耗的耐久值（默认值为 1）。
- **饱食度消耗**：设置范围挖掘时是否消耗额外的饱食度（默认值为 1）。
- **特殊锤子功能**：可单独调整特殊锤子的各项效果参数。

## 🔨 基础锤子

这些锤子遵循原版的工具，可通过对应材料在工作台合成。

- 木锤
- 铜锤
- 铁锤
- 金锤
- 钻石锤
- 下界合金锤

## ✨ 特殊锤子
这些锤子拥有独特的机制，为游戏玩法带来更多可能性。

**下界之星锤**：当锤子耐久度较低时，将提升攻击力与挖掘速度；

**海洋之心锤**：当玩家处于水中时，可以无视水下挖掘速度惩罚，如同在陆地上一样快速挖掘；

**潮涌之锤**：当玩家处于水中时，将获得"潮涌能量"状态效果；

**末影锤**：使用这把锤子挖掘的方块，其掉落物将直接传送到玩家的背包中；

**岩浆锤**：挖掘的方块会直接以烧制的成品形式掉落；

**活塞锤**：攻击目标时，会附加极高的击退效果；

**玻璃锤**：虽然耐久度仅为1点，但拥有极高的攻击力。一击必杀，一击报废；

**幽匿锤**：被它挖掘的方块会直接转化为经验球。是快速获取经验的利器。

**绿宝石锤**：挖掘或攻击时根据耐久度给予幸运效果。
